## Utilizando o GITHUB

**GITHUB do projeto:** [Terra em Bytes - Wiki](https://github.com/terraembytes/Wiki)

### Passo a passo

1. Clone o repositório na sua máquina;

    > git clone https://github.com/terraembytes/Wiki.git

    - A branch padrão do projeto se chama **main**;
    - O remote é **origin**;

2. Faça as edições e atualizações no projeto e commite elas;
    - Recomenda-se criar uma nova branch para fazer as edições, commita-las e depois fazer o merge;

    > git checkout -b nome-da-nova-branch
    >
    > git add .
    >
    > git commit -m "mensagem do commit"

3. Faça o push dos commit;

    > git push origin nome-da-nova-branch

4. No GITHUB, é criado um merge request para os demais membros, caso aprovado, o GITHUB vai mesclar as alterações na branch principal (main).
