# RTKLIB - Introdução

Este documento tem como objetivo auxiliar no processo de instalação do software RTKLIB.



### Instalação 

Na página oficial do rtklib, com link disponível abaixo, será mostrada uma tabela com várias versões e as respectivas datas que foram lançadas, tanto para Windows como para outros. As versões terminam no ano de 2020.

>https://rtklib.com/ 

Para uma versão mais nova, existe o link de RTKExplorer:

> https://github.com/rtklibexplorer/RTKLIB/releases

(ou, para entrar nessa mesma página, pesquise na aba RTKExplorer e entre no ícone Dowloads).



Depois, vá na última versão que existe a aba Assets< demo5, como por exemplo a imagem a seguir:

![image-20260106141114056](/assets/instalaçãoRTKLIB.jpg)

Obs: Ao instalar o arquivo demo5 obtemos uma versão para Windows, para Linux deve ser baixado o arquivo **targ.gz** e compilá-lo.

- Depois de baixado arquivo .zip, deve-se descompactá-lo;

- Abra o RTKLaunch;
- Clique com o botão direito no arquivo e escolha a opção "criar atalho", ao fazer isso o RTKLaunch aparece automaticamente no seu Desktop;
- Se deseja esse atalho na barra de tarefas, basta arrastar o ícone do Desktop para a barra de tarefas.
